# Recitronic - Reciclaje Electrónico

Plataforma web de Recitronic para promover el reciclaje responsable de dispositivos electrónicos.

## Descripción

Recitronic es un servicio de reciclaje electrónico que recolecta y recicla responsablemente todo tipo de dispositivos electrónicos, desde celulares y computadores hasta electrodomésticos.

## Características

- **Servicio de retiro a domicilio**: Recolectamos en tu hogar u oficina
- **Reciclaje certificado**: Proceso 100% ecológico
- **Múltiples métodos de pago**: Transferencia, tarjeta o efectivo
- **Tarifas competitivas**: Basadas en volumen de material

## Artículos que reciclamos

- Celulares y tablets
- Computadores y periféricos
- Electrodomésticos
- Otros dispositivos electrónicos

## Tecnologías utilizadas

- HTML5
- CSS3
- Bootstrap 5.3.7
- Bootstrap Icons
- jQuery 3.7.1
- Google Fonts (Sansation)
